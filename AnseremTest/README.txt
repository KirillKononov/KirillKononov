Необходимо реализовать клиент-серверное приложение: 
db mssql, backend c#, frontend html, css для «браузерной» работы с данными простой базы данных.

•	Реализовать структуру БД:
	Продажа, Контрагент (должна быть реализована таблица, содержащая Город), Контакт (клиент-контактное лицо, ответственный за продажу)
•	Реализовать серверное приложение, обрабатывая необходимые клиенту запросы: добавление, чтение, изменение, удаление записей.

•	Реализовать клиентскую часть приложения:
o	Вывод существующих продаж; столбцы:
	название продажи
	клиент-организация
	контактное лицо от организации
	ответственный за продажу
	город клиента-организации
o	Добавление новой продажи (обязательно к заполнению только «Название» продажи)
o	Изменение данных продажи прямо из списка
Кнопка «Редактировать» в каждой строке. 
o	Удаление продажи
