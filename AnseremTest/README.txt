Необходимо реализовать клиент-серверное приложение: 
DB MSSQL, Backend C#, Frontend HTML, CSS для «браузерной» работы с данными базы данных.

•	Реализовать структуру БД:
	Продажа, 
	Контрагент (должна быть реализована таблица, содержащая Город), 
	Контакт (клиент-контактное лицо, ответственный за продажу)
	
•	Реализовать серверное приложение, обрабатывая необходимые клиенту запросы: 
	добавление, чтение, изменение, удаление записей.

•	Реализовать клиентскую часть приложения:
	o	Вывод существующих продаж; столбцы:
		o	Название продажи
		o	Контрагент
		o	Контактное лицо от организации
		o	Ответственный за продажу
		o	Город контрагента
	o	Добавление новой продажи (обязательно к заполнению только «Название» продажи)
	o	Изменение данных продажи прямо из списка. Кнопка «Редактировать» в каждой строке. 
	o	Удаление продажи
